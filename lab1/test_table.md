## ⚠️ Важное примечание о методах решения

### Метод Ньютона с понижением степени (уравнения 4-й степени):
- **Возвращает:** ВСЕ 4 корня (тип `list[float | complex]`)
- **Принцип работы:**
  1. Находит первый корень r₁ методом Ньютона (касательных)
  2. Делит полином P(x) на (x - r₁) методом Горнера, получая кубический полином Q(x)
  3. Решает Q(x) = 0 методом Кардано, находя оставшиеся 3 корня
- **Особенность:** Первый найденный корень зависит от начальной точки x₀, но итоговый результат содержит все 4 корня
- **Пример:** x⁴ - 5x² + 4 = 0 → возвращает все 4 корня: [2.0, -2.0, -1.0, 1.0]

### Метод Кардано и Формула Виета:
- **Возвращают:** ВСЕ корни (тип `list[float | complex]`)
- **Примеры:**
  - x³ - 6x² + 11x - 6 = 0 → [1.0, 2.0, 3.0] (3 действительных корня)
  - x³ + 27 = 0 → [-3.0, 1.5 + 2.6i, 1.5 - 2.6i] (1 действительный + 2 комплексных)
  - x² - 4 = 0 → [-2.0, 2.0] (2 действительных корня)
