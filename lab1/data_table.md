## Примечания:

### Размеры типов данных (Python 3.9+):
- **`float`**: 8 байт (стандарт IEEE 754 double precision)
- **`complex`**: 16 байт (два float - действительная и мнимая части)
- **`int`**: 4 байта (для значений до 2³¹-1)
- **`list`**: базовый размер ~56 байт + размер элементов
- **`str`**: зависит от длины строки (~50 байт для объекта + 1 байт на символ)

### Используемые методы:

1. **Метод Ньютона с понижением степени** - для уравнений 4-ой степени (a ≠ 0)
   - Шаг 1: Находит первый корень методом касательных: x_{n+1} = x_n - f(x_n) / f'(x_n)
   - Шаг 2: Делит полином на (x - root1) методом Горнера, получая кубический полином
   - Шаг 3: Решает кубическое уравнение методом Кардано для нахождения оставшихся 3 корней
   - Возвращает все 4 корня (действительных или комплексных)

2. **Метод Кардано** - для кубических уравнений (a = 0, b ≠ 0) или после понижения степени
   - Приведение к виду: x³ + px + q = 0
   - Вычисление дискриминанта: D = (q/2)² + (p/3)³
   - Возвращает три корня (действительных или комплексных)

3. **Формула Виета** - для квадратных уравнений (a = 0, b = 0, c ≠ 0)
   - Дискриминант: D = d² - 4ck
   - Формулы корней: x = (-d ± √D) / (2c)
   - Возвращает два корня (действительных или комплексных)

4. **Линейное уравнение** - для уравнений 1-ой степени (a = 0, b = 0, c = 0, d ≠ 0)
   - Формула: x = -k / d
   - Возвращает один корень

### Особенности реализации:

- **Метод Ньютона с понижением степени**:
  - Использует 403 начальные точки для повышения надежности сходимости (0.0, x₀, плотная сетка с шагом 0.5)
  - Применяет схему Горнера (синтетическое деление) для деления полинома на (x - root1)
  - Находит все 4 корня уравнения 4-ой степени (1 через метод Ньютона + 3 через метод Кардано)
- **Метод Кардано** корректно обрабатывает все случаи дискриминанта (D > 0, D = 0, D < 0)
- **Формула Виета** использует модуль `cmath` для работы с комплексными корнями при отрицательном дискриминанте
- **Схема Горнера** для деления полинома:
  - P(x) = ax⁴ + bx³ + cx² + dx + k
  - P(x) = (x - r) × (b3x³ + c3x² + d3x + k3)
  - Коэффициенты: b3=a, c3=b+r×b3, d3=c+r×c3, k3=d+r×d3
- Программа обрабатывает вырожденные случаи (все коэффициенты равны нулю)

### Диапазоны значений:

- **Коэффициенты (a, b, c, d, k)**: Указан технический диапазон типа float (-1.80×10³⁰⁸ до 1.80×10³⁰⁸), так как коэффициенты могут принимать любые действительные значения
- **Точность (tolerance)**: Указан практический диапазон (1×10⁻¹⁵ до 1×10⁻³), так как значения вне этого диапазона не имеют практического смысла для численных методов
- **Итерации (max_iterations)**: Указан практический диапазон (100 до 10000), достаточный для сходимости метода Ньютона
- **Начальная точка (x0)**: Указан технический диапазон, но программа перебирает 403 точки (0.0, x₀, затем плотную сетку от -100 до 100 с шагом 0.5)
- **Промежуточные и выходные значения**: Указан технический диапазон float, так как результаты могут быть любыми в зависимости от входных данных
