## Примечания

### Размеры типов данных (для отчёта):
- **`float`**: 8 байт (IEEE 754 double precision)
- **`int`**: 4 байта
- **`list`**: в таблице объём оценён по полезной нагрузке (число элементов × размер элемента), как в примере из `lab1/data_table.md`

### Критерий остановки:
- Итерации прекращаются, когда \( \max_i |R_i| < eps \)

### Ограничения корректности входных данных:
- `A` — квадратная матрица
- `A` удовлетворяет условию диагонального преобладания: \(|a_{ii}| > \sum_{j\ne i}|a_{ij}|\) для всех \(i\)
- `det(A) ≠ 0`
- `A[i][i] ≠ 0` для всех \(i \in [0, n-1]\)
