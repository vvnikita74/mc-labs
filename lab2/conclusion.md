# Вывод

В ходе выполненной лабораторной работы было реализовано и исследовано решение систем линейных алгебраических уравнений методом релаксации. Программа выполняет чтение входных данных из файлов `A.txt` и `B.txt`, осуществляет проверку корректности исходной матрицы (квадратность, согласованность размерностей, ненулевая диагональ, диагональное преобладание) и, при выполнении условий применимости метода, итерационно вычисляет приближённое решение \(x\) с заданной точностью. Итогом работы является вывод найденных корней \(x_1,\dots,x_n\) и числа итераций, необходимых для достижения критерия остановки.

Работоспособность программы подтверждена тестированием на различных наборах данных, включая корректные случаи (с известными корнями) и проверки обработки типовых ошибок ввода с выводом соответствующих сообщений.
